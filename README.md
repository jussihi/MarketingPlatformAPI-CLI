# Install

## Run
```
python install.py
```

# Usage

## 1. Enable venv:

Linux:
```
source .venv/bin/activate
```

Windows (powershell):
```
.venv/Scripts/Activate.ps1
```

Windows (cmd):
```
..venv/Scripts/activate
```

## 2. Now you can run the program inside venv:
```
python mp_api_cli.py -h
```
